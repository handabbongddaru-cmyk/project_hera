# project_hera
## 실행 방법

본 프로젝트는 data/raw 및 data/processed 경로를 기준으로 구성되어 있습니다.

노트북 실행 시, 해당 경로에 맞는 데이터 파일이 필요합니다.

※ 대용량 원본 데이터는 GitHub에 포함되지 않았으며, 전처리 결과 데이터만 포함되어 있습니다.
원본 데이터는 아래 링크에서 확인 가능합니다.
https://drive.google.com/drive/folders/1IUP1aE7xcw5JGadWuuZEt5MVPcmikvm7?usp=sharing


## 데이터 구성

### 무신사뷰티 데이터
- musinsa_suncare_male.csv: 남성 선케어 제품 랭킹 데이터 (상위 제품 기준)
- musinsa_suncare_female.csv: 여성 선케어 제품 랭킹 데이터 (상위 제품 기준)
- musinsa_all_reviews.csv: 무신사뷰티 선케어 제품 전체 리뷰 데이터
- 무신사뷰티_톤업제품_후기_키워드.csv: 리뷰 텍스트 기반 주요 키워드 추출 및 빈도 분석 결과 데이터
- 무신사뷰티_상세페이지_완료.csv: 상세페이지 이미지 기반 제품 소구 포인트 및 기능 정보 추출 데이터
  
- 무신사뷰티_톤업제품_후기키워드_병합.csv: 키워드 데이터 전처리 및 병합을 통해 분석에 최종적으로 사용된 데이터

### 헤라 데이터
- 헤라_리뷰_최종_전체_23367개.csv: 헤라 선케어 제품 전체 리뷰 원본 데이터
- 헤라_v5.csv: 리뷰 텍스트 기반 주요 키워드 추출 데이터
- 헤라_v6.csv: 리뷰 감성 분석 결과 데이터 (긍정/부정/중립 분류)

- 헤라_v8.csv: 전처리 및 분석에 최종적으로 사용된 통합 데이터

### 설문 데이터
- 선케어_설문_응답.csv: 남성 대상 선케어 및 톤업 제품 인식 설문 원본 데이터
  
- 선케어_설문_응답_v2.csv: 결측 처리 및 응답 값 정제 완료된 전처리 데이터
